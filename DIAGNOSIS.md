# DIAGNOSIS.md - 诊断报告模板

## 触发条件

- 初次初始化完成后
- 使用者说"给我出本周复盘"
- 周复盘跌入C/F区
- 使用者主动要求诊断

## 输出格式

```

Agent：[名] | 生成时间：[日期] | 类型：[初次诊断/周复盘]

|------|------|------|
| ✅ 描述清晰可操作 | X条 | 有具体检查方式，可直接执行 |
| ⚠️ 描述模糊需改写 | X条 | 有记录但无法指导具体行为 |
| ❌ 缺少检查方式   | X条 | 记录不完整，需补充 |
| 📦 归档区        | X条 | 超过30天未触发 |
```
