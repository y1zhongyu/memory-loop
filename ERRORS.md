# ERRORS.md - 错误清单

## 使用说明

- **活跃区**：30天内触发过的错误，建议任务前读取
- **归档区**：超过30天未重复，保留历史但不占日常token
- 不限条数，按时间自动流转
- 重复犯时从归档区移回活跃区，更新重复次数，强制重写检查方式

## 记录模板

```
#### 错误记录 #XXX
错误类型：[信息遗漏/事实错误/格式违规/流程跳步/沟通缺失]
触发场景：[做什么任务时发生]
错误行为：[做了什么]
正确行为：[应该做什么]
检查方式：[下次如何自我验证，必填，写不出来说明描述不够具体]
首次日期：[日期]
最近触发：[日期]
重复次数：[0]
```
